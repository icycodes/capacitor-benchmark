| Symbol | Description |
|---|---|
| `ApiResponse<T>` | Typed wrapper: `data: T`, `status: number`, `headers: Record<string, string>` |
| `UnauthorizedError` | Extends `Error`; carries `status: 401` and `url: string`; uses `Object.setPrototypeOf` for safe `instanceof` checks across transpilation boundaries |
| `httpGet<T>` | Injects `Authorization` header, calls `CapacitorHttp.request({ method: 'GET', … })`, passes `params` verbatim, no `Content-Type` |
| `httpPost<T, B>` | Injects `Authorization` header, assigns `body` to `data`, adds `Content-Type: application/json` only when `body` is defined |

**Shared private logic** (`buildAuthHeaders` + `executeRequest`):
- Reads `auth_token` from `Preferences`; skips the header when the value is `null` or `""`.
- On **401**: calls `Preferences.remove({ key: 'auth_token' })` then throws `UnauthorizedError`.
- On **other non-2xx**: throws `Error` with message `HTTP <status>` without clearing preferences.
- On **2xx**: returns `ApiResponse<T>` with the raw `response.data` cast to `T`.
