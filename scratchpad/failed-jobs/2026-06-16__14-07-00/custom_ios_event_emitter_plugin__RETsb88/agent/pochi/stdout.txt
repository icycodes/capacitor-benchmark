The product team wants to push a periodic `tick` event from native Swift code to the JavaScript layer once per interval, so the web UI can drive an animation without writing its own timer. No plugin exists yet for this capability. Your job is to build a Capacitor v8 local plugin in Swift that uses `Foundation.Timer` to emit a custom event to JavaScript using `notifyListeners`, register the plugin instance with the Capacitor bridge in the existing view controller, and expose it to the web layer through a TypeScript wrapper that listens for the events.
